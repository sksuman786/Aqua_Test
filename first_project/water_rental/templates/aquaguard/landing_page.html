Frequently Asked Questions

Everything you need to know about AquaGuard

How accurate are the sensor readings? ▼
AquaGuard uses calibrated IoT sensors with 99.5% accuracy. All readings are validated against certified lab standards. Sensors are automatically calibrated monthly to ensure precision.
What water parameters are monitored? ▼
We monitor 15+ parameters including pH, temperature, turbidity, conductivity, dissolved oxygen, chlorine, nitrate, phosphate, and can detect bacterial and chemical contaminants. Custom parameters are available for enterprise plans.
How quickly can I get set up? ▼
Basic setup takes 2-4 hours. Our technical team handles installation and calibration. You can start receiving data within 24 hours of installation. Express setup is available for an additional fee.
Is there API access for integration? ▼
Yes! Professional and Enterprise plans include REST API access. We provide webhooks for real-time data integration with your existing systems. Full API documentation is available.
Do you offer a free trial? ▼
Yes! We offer a 14-day free trial with full access to Professional features. No credit card required, and you can cancel anytime. Setup is included at no cost.
